SPY LSTM Forecasting (FITE Project)

Time-series forecasting project to predict next-day SPY adjusted close using LSTM models.

Python TensorFlow LSTM yfinance scikit-learn

Project Flow

  • Downloaded SPY data and engineered indicators: MA10/30/90, log returns, volatility, and RSI14.
  • Prepared sequential samples and trained three LSTM models with 90-day, 30-day, and 10-day lookback windows.
  • Evaluated model performance using MSE, RMSE, and MAE.
  • Benchmarked model outputs against a MA10 baseline and plotted predictions against actual values.
